# Phase 07：多商户试用版 / 商户 Beta

## 阶段目标

让 1-3 个外部商户能试用系统，验证是否对别人也有提效价值。

## 本阶段 Codex 做什么

- 商户注册/创建
- 初始化流程
- 设备导入
- 订单导入
- 商户中心
- 教程和反馈入口
- 试用开关

## 本阶段你做什么

- 找 1-3 个商户试用
- 帮他们导入设备
- 收集反馈
- 判断收费价值

## 本阶段交付物

- 商户初始化流程
- 外部试用反馈
- Beta 问题清单
- 下一版计划

## 进入下一阶段条件

- 至少 1 个外部商户完整试用
- 数据隔离无误
- 外部商户完成核心流程
- 反馈已整理

## 执行原则

- 先计划，后执行。
- 不跨阶段。
- 不破坏现有可用流程。
- 结果写入本阶段 `artifacts/`。
