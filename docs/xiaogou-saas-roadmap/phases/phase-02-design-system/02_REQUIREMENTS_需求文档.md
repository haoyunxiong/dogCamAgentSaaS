# Phase 02 需求文档：设计系统与 Figma 基础组件

## 业务背景

本项目当前正在从内部租赁助手升级为租赁商户运营标准化系统。本阶段用于完成：建立稳定 Design System 和 Figma 组件库，避免后续 UI 反复推翻。

## 功能需求

- 通过 Figma MCP 创建设计系统
- 创建颜色变量/字体样式/组件
- 创建移动端核心页面初稿
- 创建 PC 端后台基础框架
- 输出设计验收报告

## 用户侧需求

- 提供参考产品
- 确认主色和品牌风格
- 审核 Figma 初稿
- 确认以后不大幅换风格

## 输出要求

- Figma 设计系统
- 组件库
- 移动端核心页面初稿
- PC 端框架初稿
