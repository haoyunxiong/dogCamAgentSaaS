# Phase 05：员工 SOP 与新员工上手系统

## 阶段目标

把业务操作做成任务、检查清单和引导，让新员工快速上手。

## 本阶段 Codex 做什么

- 创建任务中心
- 创建发货检查清单
- 创建归还验机清单
- 创建异常处理提示
- 创建常用话术库
- 创建操作日志

## 本阶段你做什么

- 提供现有培训流程
- 提供检查清单内容
- 提供话术
- 让新员工测试

## 本阶段交付物

- 任务中心
- SOP 检查清单
- 话术库
- 新员工测试反馈

## 进入下一阶段条件

- 新员工能独立处理一次订单
- 检查清单可用
- 话术可用
- 不同角色任务不同

## 执行原则

- 先计划，后执行。
- 不跨阶段。
- 不破坏现有可用流程。
- 结果写入本阶段 `artifacts/`。
