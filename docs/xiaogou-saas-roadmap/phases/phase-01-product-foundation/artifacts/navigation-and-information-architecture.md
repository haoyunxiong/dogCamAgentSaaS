# Phase 01 导航与信息架构确认

确认日期：2026-06-12

## 总原则

手机端是移动运营工作台，PC 端是完整管理后台。

不把手机端做成 PC 缩小版，也不把 PC 端做成手机放大版。

## 手机端导航

手机端底部导航固定为：

1. 工作台
2. 订单
3. 档期
4. 设备
5. 我的

## 手机端重点任务

| 导航 | 主要任务 |
|---|---|
| 工作台 | 今日待办、快捷入口、异常提醒、免押审核入口、询单快捷入口 |
| 订单 | 快速处理订单、查看状态、进入订单详情 |
| 档期 | 快速查档期、判断设备可用性 |
| 设备 | 设备快查、查看设备状态 |
| 我的 | 员工个人入口、账号、基础设置 |

## 手机端不放入底部主导航的能力

| 能力 | 处理方式 |
|---|---|
| 免押 | 工作台快捷入口、订单详情入口、PC 独立模块 |
| 询单 | 工作台快捷入口或订单流程入口 |
| 报表 | PC 为主，手机端后续只做轻量概览 |
| 商户协作 | 未来预留，不进入当前阶段 |

## PC 端导航

PC 端主导航固定为：

1. 工作台
2. 询单中心
3. 订单中心
4. 档期中心
5. 设备中心
6. 客户中心
7. 免押管理
8. 物流发货
9. 财务结算
10. 报表中心
11. 设置中心

## PC 端模块说明

| 导航 | 模块定位 |
|---|---|
| 工作台 | 经营概览、今日待办、异常提醒 |
| 询单中心 | 询单记录、查档期、报价、转订单 |
| 订单中心 | 订单列表、订单状态、批量处理 |
| 档期中心 | 设备占用、空闲查询、租期安排 |
| 设备中心 | 型号、单台设备、设备状态、资产管理 |
| 客户中心 | 客户资料、订单历史、沟通记录 |
| 免押管理 | 免押申请、押金状态、规则查看 |
| 物流发货 | 发货、运单、物流追踪、批量寄件 |
| 财务结算 | 租金、押金、退款、成本、结算 |
| 报表中心 | 收入、设备效率、渠道、经营分析 |
| 设置中心 | 基础设置、渠道配置、商户配置预留 |
