# Phase 01 页面清单

确认日期：2026-06-12

## 页面清单用途

本文件固定 Phase 01 的页面范围和页面定位，不代表已经开发页面或改造路由。

## 手机端页面清单

| 页面 | 导航归属 | 主要用途 | 备注 |
|---|---|---|---|
| 工作台 | 工作台 | 今日待办、快捷入口、异常提醒 | 手机端第一入口 |
| 订单列表 | 订单 | 查看订单、筛选状态、快速进入详情 | 高频轻操作 |
| 订单详情 | 订单 | 发货、归还、免押、物流等订单动作入口 | 不独立放入底部导航 |
| 档期查询 | 档期 | 快速查档期、判断设备可用性 | 支持运营接单判断 |
| 设备快查 | 设备 | 查看设备状态、编号、可租情况 | 后续可接扫码 |
| 我的 | 我的 | 员工个人入口、账号、基础设置 | 不承载复杂后台设置 |

## 手机端快捷入口

| 快捷入口 | 放置建议 |
|---|---|
| 询单 | 工作台快捷入口或订单流程入口 |
| 免押审核 | 工作台快捷入口、订单详情入口 |
| 发货确认 | 工作台待办、订单详情 |
| 归还提醒 | 工作台待办、订单详情 |
| 设备扫码 / 快查 | 工作台快捷入口、设备页 |

## Legacy / 渠道页面处理

| 旧能力 / 页面方向 | Phase 01 处理 |

| 闲鱼账号连接 | 降级为 legacy channel 配置，不作为主入口 |
| 打开闲鱼 IM | 可作为渠道入口候选，不作为工作台主动作 |

| 市场监控 | 标记 legacy / 暂缓，不进入主线 |

| 学习审核 | 标记 legacy 客服训练能力，不进入主线 |
| 人工接管 | 后续可评估并入员工待办 / SOP |
