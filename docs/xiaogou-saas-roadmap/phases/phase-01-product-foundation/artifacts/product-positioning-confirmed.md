# Phase 01 产品定位确认版

确认日期：2026-06-12

## 产品名称

对外名称暂定为：

```text
小狗相机助手
```

产品版本 / 后台名称为：

```text
小狗相机助手商户版
```

## 一句话定位

小狗相机助手商户版是面向租赁商家的运营标准化系统，用于规范和提效商户内部的询单、订单、档期、设备、免押/押金、物流、归还、验机、员工协作和经营数据管理流程。

## 当前不是

- 不是闲鱼助手。
- 不是闲鱼自动回复工具。
- 不是二手设备交易平台。
- 不是商户之间调货平台。
- 不是当前阶段的外部商户协作网络。

## 行业聚焦范围

Phase 01 先聚焦：

```text
相机、运动相机、CCD、摄影器材、数码设备、演唱会/旅游拍摄设备租赁商家。
```

当前主线是：

```text
先把影像 / 数码设备租赁商户内部经营流程标准化，再预留未来行业扩展。
```

未来可以扩展到其他设备租赁行业，但当前阶段不泛化为所有租赁行业。

## 短期目标

短期目标是：

1. 把当前内部租赁业务流程规范化。
2. 让新员工可以快速上手。
3. 让订单、档期、设备、物流、免押、报表形成标准系统。
4. 让后续多商户 SaaS 化有结构基础。
5. 保留未来商户协作网络的架构预留，但不进入当前阶段实现。

## 当前阶段边界

Phase 01 只固定产品定位和系统骨架，不开发功能，不做 UI 设计系统，不操作 Figma，不做数据库迁移，不重构业务代码。

## 闲鱼定位

闲鱼只作为 legacy channel / 订单来源渠道之一。

处理原则：

- 不再作为产品核心。
- 不再作为主导航中心。
- 不再作为新架构依赖。
- 旧能力不删除，先标记 legacy / 渠道化候选 / 待确认。

## 保留原则

保留的是租赁商户运营能力，不是旧命名、旧 UI 或旧闲鱼实现方式。

必须保留并后续可去闲鱼化、渠道化、SaaS 化的能力包括：

1. 查档期。
2. 发档期 / 报价。
3. 创建订单。
4. 订单履约。
5. 待发货 / 进行中 / 待归还 / 逾期 / 完结。
6. 免押 / 押金。
7. 顺丰发货和物流追踪。
8. 归还 / 验机 / 完结。
9. 设备型号 / 单台设备。
10. 设备状态。
11. 报表统计。
12. 费用统计。
13. 配置能力。
14. 日志和运行状态。
15. 当前启动流程。
16. 现有有效数据保护。
17. 店铺 / 门店维度。
18. 知识库。
19. 商品 / 型号绑定中与租赁业务相关的能力。
