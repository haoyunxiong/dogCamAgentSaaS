# Phase UI-V2 Demo 验收标准

## 产品体验验收

- [ ] PC 端主框架方向确认。
- [ ] Mobile 端主框架方向确认。
- [ ] PC 端核心页面结构确认。
- [ ] Mobile 端核心页面结构确认。
- [ ] 视觉风格确认。
- [ ] 交互结构确认。
- [ ] mock 数据足以表达核心业务状态。

## 技术边界验收

- [ ] Demo 位于 `prototypes/ui-v2-demo`。
- [ ] Demo 只使用 mock 数据。
- [ ] Demo 不接 Electron。
- [ ] Demo 不接 Python backend。
- [ ] Demo 不接数据库。
- [ ] Demo 不接真实 API。
- [ ] Demo 不修改旧系统业务页面。

## 阶段流转验收

- [ ] 用户确认 UI-V2 Demo 可以作为后续迁移目标。
- [ ] 旧系统能力迁移范围已单独列出。
- [ ] 旧系统迁移顺序已单独调度。
