# Phase UI-V2 Demo 阶段目标

## 核心目标

创建一套独立的新产品 Demo，用于确认“小狗相机助手商户版”的未来视觉风格、PC 端管理后台体验和手机端移动工作台体验。

## 成功标准

- PC 端主框架、核心页面结构和信息密度得到确认。
- Mobile 端任务流、底部导航和高频操作得到确认。
- Demo 使用 mock 数据即可完整表达体验方向。
- 不依赖旧系统业务页面、Electron、Python、数据库或真实 API。
- 用户确认 Demo 风格可以作为后续迁移目标。

## 阶段边界

本阶段只验证新体验，不迁移旧系统能力。

旧系统能力迁移必须在 Demo 风格和结构确认后，单独调度。
