# Phase 08：商户协作网络预留

## 阶段目标

只做调货、代发、二手交易的规划和数据预留，不做交易闭环。

## 本阶段 Codex 做什么

- 输出商户协作需求文档
- 调货需求模型
- 可供设备模型
- 代发协作模型
- 二手交易模型
- 预留导航/数据表设计
- 不接支付

## 本阶段你做什么

- 收集真实调货/代发需求
- 记录群里交易流程
- 确认平台责任边界
- 确认第一版撮合方式

## 本阶段交付物

- 商户协作 PRD
- 数据模型预案
- 风险清单
- 商业化假设

## 进入下一阶段条件

- 收集 20 条真实需求
- 明确平台责任
- 明确是否担保
- 明确收费方式
- 再决定是否开发

## 执行原则

- 先计划，后执行。
- 不跨阶段。
- 不破坏现有可用流程。
- 结果写入本阶段 `artifacts/`。
