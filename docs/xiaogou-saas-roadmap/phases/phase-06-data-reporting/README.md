# Phase 06：经营数据与报表中心

## 阶段目标

让系统帮助老板看经营：收入、设备 ROI、渠道、效率、异常。

## 本阶段 Codex 做什么

- 收入统计
- 设备/型号收益
- 渠道来源统计
- 逾期统计
- 发货效率统计
- 免押通过率
- 员工效率
- 周报/月报

## 本阶段你做什么

- 确认最重要的 5 个指标
- 核对数据口径
- 判断报表价值

## 本阶段交付物

- 经营看板
- 设备收益报表
- 渠道报表
- 效率报表
- 数据口径文档

## 进入下一阶段条件

- 至少 5 个报表可用
- 数据口径清楚
- 统计结果与实际业务基本一致
- 不用 Excel 人工汇总

## 执行原则

- 先计划，后执行。
- 不跨阶段。
- 不破坏现有可用流程。
- 结果写入本阶段 `artifacts/`。
