# Phase 06 需求文档：经营数据与报表中心

## 业务背景

本项目当前正在从内部租赁助手升级为租赁商户运营标准化系统。本阶段用于完成：让系统帮助老板看经营：收入、设备 ROI、渠道、效率、异常。

## 功能需求

- 收入统计
- 设备/型号收益
- 渠道来源统计
- 逾期统计
- 发货效率统计
- 免押通过率
- 员工效率
- 周报/月报

## 用户侧需求

- 确认最重要的 5 个指标
- 核对数据口径
- 判断报表价值

## 输出要求

- 经营看板
- 设备收益报表
- 渠道报表
- 效率报表
- 数据口径文档
