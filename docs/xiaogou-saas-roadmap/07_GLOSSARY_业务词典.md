# 07 业务词典

| 词 | 定义 |
|---|---|
| 商户 Merchant | 使用系统的租赁商家主体 |
| 门店 Store | 商户下的经营/发货门店 |
| 仓库 Warehouse | 设备存放或发货位置 |
| 渠道 Channel | 订单或询单来源，如闲鱼、小红书、抖音、私域、手动录入 |
| 询单 Inquiry | 客户咨询租赁需求但未正式下单的记录 |
| 报价 Quote | 基于设备、档期、租期、押金/免押规则生成的租赁价格或可租方案 |
| 档期 Schedule | 设备在某时间范围内的占用/空闲状态 |
| 设备型号 DeviceModel | 一类设备，如 acepro2、g12、g7x2 |
| 单台设备 DeviceUnit | 具体某一台设备，带编号/SN/状态 |
| 订单 Order | 客户租赁需求正式确认后的履约记录 |
| 订单明细 OrderItem | 订单中被租赁的设备、配件、租期、价格等明细 |
| 免押 Deposit-Free | 订单免押申请或免押审核流程 |
| 押金 Deposit | 客户租赁前需支付或冻结的履约担保金额 |
| 物流 Shipment | 发货、运单、物流追踪和配送异常处理 |
| 归还验机 ReturnInspection | 客户归还设备后的检查流程 |
| 财务结算 Settlement | 租金、押金、退款、成本、结算等资金处理 |
| 报表 Report | 面向经营分析的订单、设备、收入、渠道和效率数据汇总 |
| Legacy Channel | 历史渠道或旧实现，保留兼容但不作为新产品主架构 |
| 闲鱼 Xianyu | legacy channel / 订单来源渠道之一，不再是产品核心 |
| 商户协作 | 商户之间调货、代发、供需撮合、二手交易等未来模块 |
| 配置治理 Configuration Governance | 配置来源、敏感配置、商户配置和凭证管理的治理方向，归属 Phase 03 |
