## 配置治理任务

Codex 需要：

1. 只读扫描配置来源；
2. 填写 CONFIG_SOURCE_AUDIT；
3. 标记配置分类；
4. 标记敏感风险；
5. 判断是否进入 Git；
6. 输出未来存储建议；
7. 规划 ConfigService / CredentialService；
8. 如果涉及数据库，必须先输出迁移和回滚方案；
9. 不输出真实敏感值；
10. 不在未经确认时修改业务代码。
