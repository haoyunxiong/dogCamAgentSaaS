# 01 产品定位

更新日期：2026-06-12

## 产品名称

对外名称暂定为：

```text
小狗相机助手
```

产品版本 / 后台名称为：

```text
小狗相机助手商户版
```

## 一句话定位

小狗相机助手商户版是面向租赁商家的运营标准化系统，用于规范和提效商户内部的询单、订单、档期、设备、免押/押金、物流、归还、验机、员工协作和经营数据管理流程。

## 不再做什么

- 不再以“闲鱼助手”为产品核心。
- 不再围绕闲鱼自动回复建立系统架构。
- 不再把闲鱼订单视为唯一订单来源。
- 不在当前阶段做商户之间交易闭环。
- 不在当前阶段做二手设备交易平台。

## 当前阶段重点

当前阶段的核心不是“卖给所有商户”，而是先把自己已经跑通的租赁流程做成标准系统：

```text
询单 → 查档期 → 报价/发档期 → 创建订单 → 创建免押/押金 → 发货 → 物流跟踪 → 到期提醒 → 归还验机 → 完结订单 → 数据沉淀/报表分析
```

短期目标：

1. 把当前内部租赁业务流程规范化。
2. 让新员工可以快速上手。
3. 让订单、档期、设备、物流、免押、报表形成标准系统。
4. 让后续多商户 SaaS 化有结构基础。
5. 保留未来商户协作网络的架构预留，但不进入当前阶段实现。

## 目标用户

| 用户 | 关注点 |
|---|---|
| 老板 / 超级管理员 | 收入、设备收益、订单异常、员工效率、整体经营 |
| 店长 | 今日待办、订单流转、人员分工、设备调度 |
| 运营 | 询单、查档期、报价、创建订单、客户沟通 |
| 发货员 | 发货、物流、归还、验机、设备状态 |
| 财务 | 租金、押金、退款、结算、报表 |
| 新员工 / 客服运营 | SOP、话术、业务流程上手；先作为使用场景角色，不作为 Phase 01 正式权限角色 |

## 初期行业边界

先聚焦：

```text
相机、运动相机、CCD、摄影器材、数码设备、演唱会/旅游拍摄设备租赁商家
```

未来可扩展到其他租赁行业，但不要在第一阶段做得过泛。

当前主线是：

```text
先把影像 / 数码设备租赁商户内部经营流程标准化，再预留未来行业扩展。
```

## 闲鱼定位

闲鱼只作为 legacy channel / 订单来源渠道之一。

处理原则：

1. 不删除旧逻辑。
2. 不重构旧逻辑。
3. 不进入新产品主导航。
4. 不作为新架构核心。
5. 只在文档中标记为 legacy / 渠道化候选 / 待确认。
