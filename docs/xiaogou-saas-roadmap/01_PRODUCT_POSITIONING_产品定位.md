# 01 产品定位

## 一句话定位

小狗相机助手商户版是面向租赁商家的运营标准化系统，帮助商家管理订单、档期、设备、免押、物流、员工协作和经营数据。

## 不再做什么

- 不再以“闲鱼助手”为产品核心。
- 不再围绕闲鱼自动回复建立系统架构。
- 不再把闲鱼订单视为唯一订单来源。
- 不在当前阶段做商户之间交易闭环。

## 当前阶段重点

当前阶段的核心不是“卖给所有商户”，而是先把自己已经跑通的租赁流程做成标准系统：

```text
询单 → 查档期 → 报价/发档期 → 创建订单 → 创建免押/押金 → 发货 → 物流跟踪 → 到期提醒 → 归还验机 → 完结 → 数据沉淀
```

## 目标用户

| 用户 | 关注点 |
|---|---|
| 老板 | 收入、设备收益、订单异常、员工效率、整体经营 |
| 店长 | 今日待办、订单流转、人员分工、设备调度 |
| 运营 | 询单、查档期、报价、创建订单、客户沟通 |
| 发货员 | 发货、物流、归还、验机、设备状态 |
| 财务 | 租金、押金、退款、结算、报表 |

## 初期行业边界

先聚焦：

```text
相机、运动相机、CCD、无人机、稳定器、摄影器材、演唱会/旅游拍摄设备租赁商家
```

未来可扩展到其他租赁行业，但不要在第一阶段做得过泛。
