# 小狗相机助手品牌与企业级 UI 设计

## 背景

当前应用是 Electron + Vue 桌面工作台，已有租赁运营、订单履约、顺丰寄件、人工接管、学习审核、知识库和设置等模块。本次需求是把“闲鱼助手 Pro / XianyuAgentPro”改为“小狗相机助手”，并把后续全局 UI 优化方向定为企业级运营工作台，同时说明如何部署到云服务并绑定固定域名。

## 范围

本轮不把 Electron 改成 Web SaaS，不修改 Python、IPC、数据库结构、登录、消息同步、自动回复、顺丰和飞书业务逻辑。

本轮实施范围：

- 前端可见品牌文案改为“小狗相机助手”。
- 壳层文案调整为相机租赁运营语境。
- 写入企业级 UI 后续改造方向。
- 新增云部署和固定域名说明文档。

## 视觉方向

采用用户确认的 A+C 方向：

- A 作为全局基础：深色克制 Sidebar、白色内容区、稳定浅灰背景、青绿色主操作色、紧凑 Topbar、状态 Badge、表格优先。
- C 作为高频页面工作流：订单履约、订单详情、顺丰寄件、人工接管等页面保留列表上下文，复杂详情使用右侧 Drawer 或右侧摘要栏。

## 品牌文案

统一主品牌：

```text
小狗相机助手
```

建议副标题：

```text
相机租赁运营工作台
```

保留业务语境中的“闲鱼”作为渠道名称，例如“闲鱼账号登录”“打开闲鱼 IM”“闲鱼消息同步”。这些不是产品品牌，不应全局删除。

## 实施分期

### 第一阶段：品牌与壳层文案

修改前端可见品牌、浏览器标题、package 描述、Sidebar 品牌区、Topbar 搜索提示和订单导航命名。此阶段不改打包 productName，避免触发启动器同步和可执行文件名变化。

### 第二阶段：全局 UI 收敛

收敛 tokens、Sidebar、Topbar、BaseButton、BaseCard、BaseBadge、BaseTable、BaseDrawer。减少渐变、强阴影和低密度卡片。

### 第三阶段：核心页面工作台化

Dashboard 改成“今天最需要处理什么”的待办工作台；订单履约改表格；订单详情改 Drawer；顺丰寄件压缩 Stepper 并加入右侧摘要栏。

## 云部署说明边界

当前代码仍是桌面应用，不是直接可部署为公网 Web SaaS 的架构。因此本轮文档提供两条路径：

- 当前可落地路径：桌面端继续运行，云上部署 MySQL、备份、日志和可选下载页/API 域名。
- 后续 SaaS 路径：将 Vue 渲染层 Web 化，新增服务端 API、认证、租户隔离，再用 Nginx/HTTPS 绑定固定域名。

## 验收标准

- 应用壳层可见品牌显示“小狗相机助手”。
- HTML 标题显示“小狗相机助手”。
- package 描述使用“小狗相机助手”。
- 云部署文档存在，并说明固定域名绑定步骤。
- 不修改业务逻辑、IPC、数据库 schema、Python 自动回复逻辑。
